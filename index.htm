Basic demo

This simple application is a quick start demo with minimum settings.

  • Description

This demo application is quick start demo with minimum set of settings. The application demonstrates how to:

  • Filter files to display only JPEGs.
  • Upload the original file along with 120x120 thumbnail.
  • Receive files on server and put them to a destination folder.
  • Redirect to another page after successful upload.

After the upload process completes uploader redirects you to a page containing links to the uploaded files.

This demo application works with any edition of Upload Suite ( Express, Professional, and Premium).
